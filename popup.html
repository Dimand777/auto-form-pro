Help & Git Instructions ▼

Quick Start:

  1. Click Record to start capturing your actions
  2. Interact with form elements on the page
  3. Click Stop to save the scenario
  4. Select a scenario and click Play to replay

Git Setup (for developers):

  1. Initialize repository: git init
  2. Add all files: git add .
  3. Commit changes: git commit -m "Initial setup"
  4. Add remote: git remote add origin <your-repo-url>
  5. Push to GitHub: git push -u origin main
